# Hospital Appointment System

## Project Description
The Hospital Appointment System is a web-based application developed using PHP and MySQL. It helps patients book appointments with doctors online easily.

## Features
- Patient Registration
- Login System
- Doctor Appointment Booking
- Contact Page
- Database Integration
- User Authentication

## Technologies Used
- PHP
- MySQL
- HTML
- CSS

## How to Run
1. Install XAMPP
2. Place project folder inside htdocs
3. Import SQL file into phpMyAdmin
4. Start Apache and MySQL
5. Open localhost in browser
